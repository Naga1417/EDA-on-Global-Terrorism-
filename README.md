# EDA-on-Global-Terrorism-
This repository contains Python code for the analysis of global terrorism data. The dataset used for this analysis is "globalterrorismdb_0718dist.csv." The code provides various data visualizations and insights derived from the dataset.

<h2>Data Source</h2>

You can obtain the dataset used in this analysis from the link https://bit.ly/2TK5Xn5

<h3>Setup</h3>

Before running the code, make sure you have the following Python libraries installed:

pandas
numpy
matplotlib
seaborn
<h3>You can install these libraries using pip:</h3>

pip install pandas numpy matplotlib seaborn

<h1>Key Observations</h1>

<h3>Maximum Casualties</h3>

Maximum number of people killed in an event: 1570

Maximum number of people wounded in an event: 8191

<h3>Country-wise Attacks</h3>

The top 15 countries with the most attacks are visualized in a bar chart.

<h3>Terrorist Activities by Region in Each Year</h3>

A time series analysis of terrorist activities by region is visualized in an area chart.

<h3>Number of Terrorist Activities Each Year</h3>

The number of terrorist activities in each year is visualized in a countplot.

<h3>Methods of Attack</h3>

The most common methods of attack are visualized.

<h3>Type of Attack</h3>

The most common types of attack are visualized.

<h3>Most Active Terrorist Organizations</h3>

The most active terrorist organizations are visualized in a bar chart.

<h3>Global Distribution of Terrorist Attacks</h3>

A scatter plot of the global distribution of terrorist attacks is created using latitude and longitude data.

<h3>Tableau Dashboard</h3>

Additionally, a Tableau dashboard has been created to provide an interactive and comprehensive view of the data. You can access the dashboard for further analysis and exploration.

<h2>Insights</h2>

1.Attacks have increased over time, with a significant increase in the number of casualties.

2.Iraq has experienced the most attacks.

3.The Middle East and North Africa regions are the most targeted.

4.Bombing/Explosions are the most common methods of attack.

5.Private citizens and property are the most common targets.

6.The "Taliban" and "ISIL" terrorist groups are among the most active organizations.

7.For more detailed insights and visualizations, refer to the code and Tableau dashboard in this repository.
